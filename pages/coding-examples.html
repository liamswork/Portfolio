Fancy buttons coloured with a mixin by classname.

Start by creating the mixin to colour objects by their class.

This mixin, when initialised will take an optional parameter, this is to access any of the layers of our colour map for whatever reason we choose to use a colour from the darkened palette.

It then loops through each key of the colour map, and attempts to style objects that contain that key name.

The keys are themes such as facebook blue, github purple, or related to coding, such as websites, which have their own assigned blue colour. This blue is assigned to our project tooltips to distinguish those types of projects from others.

Objects such as the buttons, have their own themes based on the icons class within them. Looking below, we can see the classes of "facebook" and "linkedin" as examples. As previously mentioned, we have colours associated with these words.

Using 2 more mixins, I added a glowing border to these buttons, and on hover, extended this glow by passing the keyword of "hover"

We are now able to reproduce our buttons with minimal effort. The only other part we would need to add is the HTML for new buttons, and specific selectors in the class colour mixin.

Above is the basic levitate mixin, and glowborder mixin. I have only included the 'all' argument to keep it short, but accepts the words 'left' 'right' 'top' 'bottom' also.
